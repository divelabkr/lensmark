## 🔒 정직성 규칙 (제품 가드레일 = 마케팅 규칙 · 취향 아님)

차별화가 곧 **신뢰**라서, 과장은 곧 자기파괴다. 모든 자산은 아래를 지킨다.

- ❌ 수익·재배성공 **보장** 금지
- ❌ **단일값** 과장 금지 → 항상 **P10·P50·P90 범위**
- ❌ **토지 매입 추천** 금지 (작물·소득만)
- ❌ "검증된 소득" 표현 금지 *(현재 소득 base는 데모·미검증)*
- ✅ 톤 = **"추정·참고"**, 겸손, "덜 무모하게 결정하도록 돕는 무료 도구"
- ✅ 면책·base 출처·연도 노출

## 현재 상태 고지 (정직)

- **무료 베타.**
- **소득 base = 데모(verified:false).** 실 RDA 소득자료 적재 전이라 앱이 "추정"을 강제한다.
- → 지금은 **"검증된 정확도"가 아니라 "정직함·접근성·구조·축적설계"** 가 차별점이다.
