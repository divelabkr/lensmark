# LENSMARK 웹·에뮬레이터 QA — Antigravity 실행 프롬프트

> Antigravity 채팅에 아래 블록을 그대로 붙여넣어 실행. (Chrome 웹 + Android 에뮬레이터 자동 테스트)

---

너는 **LENSMARK**(현실 기반 작물·수확·소득 시뮬레이터 — 무의존성 Node http 서버 + 단일파일 PWA 대시보드)의 QA 엔지니어다.
아래 환경을 기동하고 **Chrome(웹)과 Android 에뮬레이터**에서 핵심 사용자 흐름 + 최근 변경을 검증한 뒤, 결함을 심각도별로 보고하라.
이 빌드는 **실제 API 키 없이 무료 베타 모드**로 동작하며, **dev 환경은 인증코드·로그인 링크를 화면에 노출**한다(정상 동작이니 그걸로 진행).

## 1) 테스트 환경
- **A. Chrome 데스크톱(웹)** — `http://localhost:8787/app`. localhost=**보안 컨텍스트** → 서비스워커/PWA 설치까지 검증 가능.
- **B. Android 에뮬레이터**(`emulator-5554`) — 에뮬 Chrome에서 `http://10.0.2.2:8787/app`.
  - adb PATH: `cd /Users/yongj/Documents/Playground/android-emulator && source ./use-android-emulator-env.sh`
  - 앱 열기: `adb -s emulator-5554 shell am start -a android.intent.action.VIEW -d "http://10.0.2.2:8787/app" com.android.chrome`
  - 스크린샷: `adb -s emulator-5554 exec-out screencap -p > /tmp/lm-shot.png`
  - ⚠ `10.0.2.2`는 **비보안 컨텍스트** → **SW/PWA 설치는 검증 불가**(UI·반응형·기능만). 탭 좌표는 `wm size`(1080x2400)로 환산.
- **C. 실기기**(연결돼 있으면) — `adb -s <DEVICE> reverse tcp:8787 tcp:8787` 후 `http://localhost:8787/app` → 보안 컨텍스트(SW/PWA 설치 검증 가능).

## 3) 핵심 사용자 흐름 (기능·인수)
1. **첫 방문 웰컴 온보딩**: 3단계 흐름 + CTA(📍 내 위치 / 🧭 귀농 자가진단 / 지도 둘러보기) 렌더 → 닫기 → 지도 인터랙티브. (재방문 시 안 뜨고, 신버전이면 변경점 팝업)
2. **땅 선택**: 지도 핀(필지 줌) 탭 → 토지분류 마커색 → 무료 작물 추천 칩. 시·도/시·군 줌에서도 추천 동작.
3. **무료 추천 → 작물 칩 클릭 → 정밀 소득 시뮬**(무료베타라 결제 없이 실행): **P10/P50/P90 + 근거(요인 리스트) + 신뢰도 등급 + 면책** 표시. ⚠ 보장값/단일 점추정/토지매입 추천이 있으면 결함.
4. **지도 형태 토글**: 위성/일반/지형 전환.
5. **귀농 자가진단**: 폼 입력 → 준비도 카드(밴드/축별 코멘트).
6. **재배일지**: 시작 → 작업(파종/시비/방제) 추가 → 수확 기록 → 시즌 리포트 → 삭제(파기) 동작.
7. **부가 패널**: 예산·정착비용 / 온난화 시나리오 / 출하시세 / 일일 모니터링 / 지원금 / 재배가이드 렌더.

## 4) 최근 변경 집중 검증 (이번 사이클)
- **로그인(세션=httpOnly 쿠키, S5)**: 헤더 `로그인` → 모달에 **`📱 휴대폰` / `✉️ 이메일` 탭**.
  - 휴대폰: 번호 입력 → `인증번호 받기` → 화면에 표시되는 **"개발 모드 인증번호 6자리"** 입력 → 로그인 → 버튼이 **`👤 계정`**으로.
  - 이메일: 주소 입력 → `로그인 링크 받기` → **"개발 모드 — 로그인 링크 열기"** 링크 클릭 → 자동 로그인. ⚠ 클릭 후 **주소창의 `?lm_login` 토큰이 즉시 사라져야 함**(히스토리·잔류 금지).
  - 로그아웃 → `로그인`으로 복귀. (세션 토큰은 JS/localStorage에 없어야 함 = httpOnly)
- **익명→계정 이관**: 비로그인으로 재배일지 1건 생성 → 로그인 → 그 일지가 계정에 연결되어 보이는지(이관).
- **웹푸시(M1)**: 알림 모달 → **`🔔 이 브라우저로 알림 받기`** → VAPID 미설정이라 **"준비 중"** 정직 안내가 떠야 함(실제 구독 시도 안 함 = 정상).
- **모바일 헤더(U1)**: 폭 ≤600px에서 보조 액션(저장·불러오기·PDF·공유·초기화)이 **`⋯ 더보기`** 드롭다운으로 묶이는지.
- **핀 분석 속도(U2)**: 필지 핀 탭 시 추천·지형·필지가 **동시 로드**(지연 짧음). 빠르게 다른 핀을 연속 탭해도 **결과가 섞이지 않아야**(경쟁조건).
- **PWA(A/C 환경 전용)**: manifest·서비스워커 등록, Chrome `⋮`에 **`앱 설치`/`홈 화면에 추가`** 노출, 설치 후 standalone 실행. (B 에뮬은 비보안이라 제외)
