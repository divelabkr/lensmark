## 현재 컨셉 명세 (이 기준과 코드를 비교)
- **C1** 지도가 메인 화면(풀스크린). 줌/팬으로 전국→필지까지 끊김없이 탐색(SpaceEngine식).
- **C2** 지도 위 핀/지역 클릭 → 그 위치 시뮬레이터가 패널/드로어로 등장: 무료 작물추천 → 유료 정밀 소득시뮬(P10·P50·P90) + 지형6축 + 손익분기 + 근거 + 출처·면책.
- **C3** 지도 위 이벤트(가격↑↓·서리·수요·위성갱신)가 해당 좌표에 펄스로 뜨고 LIVE 피드에 쌓임. 클릭 시 해당 필지로 이동/시뮬 오픈.
- **C4** 실데이터 경로: 배경=VWorld 위성/항공 타일(z5~19), 필지=VWorld WFS 경계, 지형=DEM→경사/향/표고, 가격=KAMIS, 기후/서리=KMA, 소득=RDA. 좌표 정합(WGS84 4326 / 타일 3857 / 필지 PNU).
- **C5** 정적 "리포트 대시보드"는 보조 뷰(폴리시만, 메인 아님).
- **C6** 가드레일: 소득은 항상 P10/50/90 범위(단일값·수익보장·재배성공보장·매입추천 금지), 출처·연도, geocode 비저장, 흙토람 미사용.
