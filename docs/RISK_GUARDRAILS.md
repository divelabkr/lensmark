## 토양 API 정책

`TORAM_COMMERCIAL_PERMISSION=false` 상태에서는 흙토람/토양도/토양검정 제한 API 호출 금지.

사용 가능한 토양 자료 우선순위:

1. 고객 업로드 토양검정서
2. 고객 수동 입력
3. 현장사진/배수 정보
4. 공개 공간정보 보조
5. 글로벌 추정값은 D등급 참고
