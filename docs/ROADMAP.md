# LANSMARK Roadmap

## P0 — Skeleton Integration
- src/lansmark 추가
- 무료 후보 API 연결
- 유료 시뮬레이션 API 연결
- 기존 문구 정리

## P1 — UI
- 주소/핀/영역 입력
- 작물 후보 카드
- 유료 시뮬레이션 결과표
- 신뢰도/전제조건/리스크 표시

## P2 — Data/API
- VWorld 주소/좌표/PNU
- KMA 기상
- KAMIS/aT 가격
- 팜맵/DEM 후보 검토

## P3 — Payment/Report
- 무료 후 유료 전환
- PDF 생성
- 공유 링크

## P4 — Farm OS Extension
- 생육 단계 캘린더
- 주간 리스크
- 병해충/재난 알림
- 실제 수확/판매 피드백 보정
